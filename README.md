Zara E-Commerce Redesign
This project is a UX/UI redesign of Zara’s e-commerce website, focused on improving usability while maintaining Zara’s recognisable brand identity. The project explored how fashion e-commerce platforms can balance strong visual branding with clear navigation, efficient product discovery, and accessible interaction design.
I created the redesigns in Figma, then used Figma’s code export/embed features to help translate the designs into code-ready components. These were then prepared for integration with Claude Code into the GitHub repository. I also developed the Google API integration, supporting the project’s alternative product discovery features.
The redesign was developed through competitor analysis of Zara, H&M, and Uniqlo, alongside heuristic evaluation, SUS questionnaires, user testing, survey feedback, and UX theory. Key issues identified in Zara’s current website included complex navigation, high cognitive load, hidden menus, limited product search clarity, and reduced visibility of key interactive elements.
The final prototype aimed to simplify the shopping journey by introducing a clearer navigation structure, improved product cards, stronger visual hierarchy, fixed navigation, breadcrumb support, and an AI-supported image search concept for product discovery. The redesign retained Zara’s minimalist aesthetic through a neutral colour palette, spacious layout, large imagery, and fashion-focused visual styling.
User feedback suggested that the redesigned website improved ease of use while still feeling consistent with Zara’s brand. The project demonstrates how evidence-based UX design can improve task completion, reduce interaction effort, and create a more accessible e-commerce experience without removing the visual identity of a fashion retailer.
